How it works

XML is all you need.

TideSurf strips wrapper elements, classes, scripts and styles. What's left is clean, semantic XML that any LLM can consume.

Navigation
Raw HTML 
<div class="header">
  <nav class="main-nav"
    role="navigation">
    <ul class="nav-list">
      <li class="nav-item">
        <a href="/"

          Home
        </a>
      </li>
      <li class="nav-item">
        <a href="/about"
          class="nav-link">
          About
        </a>
      </li>
    </ul>
  </nav>
</div>

Features

Built for agents

01

DOM compression

Strips classes, wrapper divs, scripts and styles while preserving interactive elements, semantic structure and text at 100-800 tokens per page 

const state = await browser.getState()
console.log(state.xml)
// <page url="..." title="...">
//   <nav>
//     <link id="L1">Home</link>
//   </nav>
// </page>
02

12 standard tools

Navigate, click, type, scroll, extract and more. Works with any LLM that supports function calling 

await page.click("B1")
await page.type("I1", "hello world")
await page.scroll("down", 3)
await page.select("S1", "option-2")
const text = await page.extract("T1")
03

Multi-tab

Open, switch and close tabs with independent state per tab. Full browser control through one API 

await browser.newTab()
await browser.navigate("https://docs.example.com")
const tabs = await browser.listTabs()
await browser.switchTab(tabs[0].id)
await browser.closeTab(tabs[1].id)
04

Token budget

Set a max token limit and TideSurf prioritizes interactive and visible elements, pruning the rest 

const state = await browser.getState({
  maxTokens: 500
})
// Interactive elements prioritized
// Decorative content pruned
05

MCP server

Ships with a Model Context Protocol server. Drop it into Claude Code or any MCP client 

{
  "mcpServers": {
    "tidesurf": {
      "command": "bunx",
      "args": ["tidesurf", "mcp"]
    }
  }
}
06

TypeScript-first

Full type definitions, typed error classes. Built for Bun and works with Node 

import type {
  BrowserState,
  PageAction
} from "@tidesurf/core"

const config: TideSurfConfig = {
  headless: true,
  maxTokens: 800
}

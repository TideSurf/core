How it works

XML is all you need.

TideSurf strips wrapper elements, classes, scripts and styles. What's left is clean, semantic XML that any LLM can consume.

Why this matters

Every token your agent reads costs money and eats into the context window. A GitHub page that takes 84,668 tokens raw fits in just 10,968 with TideSurf — that's the difference between one page filling your context and browsing dozens of pages in a single session.

How we measure

We launch headless Chrome, navigate to each site, and compare the full rendered DOM against TideSurf's compressed XML output. Token counts use cl100k_base estimation. No cherry-picking — these are live pages, measured as-is.

Compression varies

Heavy sites like GitHub (deep nesting, SVGs, generated classes) compress dramatically. Minimal sites like Hacker News are already lean — TideSurf still strips what it can, but the gains are modest. The more bloated the source, the more TideSurf helps.

Features

Built for agents

01

DOM compression

Strips classes, wrapper divs, scripts and styles while preserving interactive elements, semantic structure and text at 100-800 tokens per page 

const state = await browser.getState()
console.log(state.xml)
// <page url="..." title="...">
//   <nav>
//     <link id="L1">Home</link>
//   </nav>
// </page>
02

18 standard tools

Navigate, click, type, scroll, extract and more. Works with any LLM that supports function calling 

await page.click("B1")
await page.type("I1", "hello world")
await page.scroll("down", 3)
await page.select("S1", "option-2")
const text = await page.extract("T1")
03

Multi-tab

Open, switch and close tabs with independent state per tab. Full browser control through one API 

await browser.newTab()
await browser.navigate("https://docs.example.com")
const tabs = await browser.listTabs()
await browser.switchTab(tabs[0].id)
await browser.closeTab(tabs[1].id)
04

Token budget

Set a max token limit and TideSurf prioritizes interactive and visible elements, pruning the rest 

const state = await browser.getState({
  maxTokens: 500
})
// Interactive elements prioritized
// Decorative content pruned
05

MCP server

Ships with a Model Context Protocol server. Drop it into Claude Code or any MCP client 

{
  "mcpServers": {
    "tidesurf": {
      "command": "bunx",
      "args": ["tidesurf", "mcp"]
    }
  }
}
// Or connect to your running Chrome:
// "args": ["tidesurf", "mcp", "--auto-connect"]
06

TypeScript-first

Full type definitions, typed error classes. Built for Bun and works with Node 

import type {
  PageState,
  ToolDefinition
} from "@tidesurf/core"

const state: PageState =
  await browser.getState()
const tools: ToolDefinition[] =
  browser.getToolDefinitions()
07

Auto Connect

Connect to an already-running Chrome instead of launching a new one. Re-use logged-in sessions, debug live pages, seamlessly hand off between manual browsing and agent control 

// Connect to your running Chrome
const browser = await TideSurf.connect()

// Or via CLI / MCP
// tidesurf mcp --auto-connect
{
  "mcpServers": {
    "tidesurf": {
      "command": "bunx",
      "args": ["tidesurf", "mcp",
               "--auto-connect"]
    }
  }
}
